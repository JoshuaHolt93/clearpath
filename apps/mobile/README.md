# ClearPath Mobile

Native iOS/Android app (Expo + Expo Router) — a first-class client of the
ClearPath FastAPI backend, sharing the `@clearpath/*` contracts with the web app.

**Phase 5 scaffolding is in place** (config, monorepo integration, secure
session storage, typed bearer API client, auth context, login → home). It is not
yet activated into the pnpm workspace, runtime-verified, or covered by CI.

👉 **Read [`MOBILE.md`](./MOBILE.md)** for the full architecture, the one-time
activation steps, the auth model, the screen roadmap, UX principles, and the
CI/verification plan.
